**製片定調：** 皮克斯短片 + 台灣 LINE 貼圖幽默。

**病毒結構：**
```
0-3s   超展開鉤子
3-12s  角色介紹+問題設定
12-30s 超展開過程快切×4
30-45s 假緊張倒數
45-55s 荒謬揭曉+結局
55-60s 品牌CTA第四面牆
```

**畫風鎖定：**
```
3D clay toy aesthetic, soft subsurface scattering,
rounded edges, Pixar-adjacent style,
warm saturated color palette, 24fps cinematic.
```

**規則：** 跳過所有角色三視圖/場景參考圖前置流程，直接出 Shot 腳本。
