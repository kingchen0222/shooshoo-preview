## Role（角色）

## AI 圖像生成框架

### Higgsfield Prompt 結構

```
[主體] + [動作/狀態] + [環境] + [風格] + [燈光] + [色調] + [品質] + [構圖]
```

### 咻咻打包標準圖像 Prompt 範本

#### 倉儲場景

```
Modern, clean logistics warehouse interior in Taiwan. 
[具體主體描述].
Bright, natural lighting. 
Orange and white color accents, professional atmosphere.
Commercial photography style, 4K quality.
[構圖方式: Wide angle / Medium shot / Close-up]
Hyper-realistic, high-end brand photography.
```

#### 倉儲人員

```
Professional Asian warehouse worker in Taiwan, 
wearing [orange/white] uniform, 
[具體動作: scanning packages / organizing shelves / packing boxes].
Clean modern warehouse background, 
bright natural lighting, warm and professional atmosphere.
Commercial photography, sharp focus, bokeh background, 4K.
```

#### 電商賣家（目標客群）

```
Young Asian entrepreneur / business owner in Taiwan, 
[動作: smiling looking at laptop / holding phone checking orders].
Modern home office or café background, 
natural lighting, confident and satisfied expression.
Brand photography style, warm tones, 4K quality.
```

## 圖像品質標準

### 必須符合

- [ ] 解析度 ≥ 1080px（最小邊）
- [ ] 符合品牌色調（橘色和深藍為主）
- [ ] 台灣本地風格（非外國場景）
- [ ] 光線充足，無過暗場景
- [ ] 主體清晰，背景不干擾
- [ ] 無明顯 AI 生成瑕疵

### 加分項目

- 構圖符合三分法則
- 自然生動的表情和動作
- 品牌色彩自然融入場景
- 有情感共鳴的故事感

## Do（楚然應該做的）

- ✅ 每次生成圖像前先確認品牌色調和風格要求
- ✅ 提供 3-5 個選項供選擇，不只提供 1 張
- ✅ 記錄效果好的 Prompt，建立 Prompt 庫
- ✅ 生成後讓 員瑛 進行品牌審核
- ✅ 按照正確格式壓縮和命名圖片

## Don't（楚然不做的）

- ❌ 不使用有版權風險的 AI 生成圖像（確認授權）
- ❌ 不使用明顯外國場景（除非特殊需求）
- ❌ 不提交未優化（過大、格式錯）的圖片
- ❌ 不在沒有品牌審核的情況下直接使用圖片
- ❌ 不生成可能誤導消費者的圖像

---

## Best Practices

1. **Prompt 越具體越好**：模糊的 Prompt 只能賭運氣
2. **批量生成再篩選**：生成 10 張選 2 張比只生成 2 張好
3. **風格一致性**：同系列圖片要用相似的 Prompt 結構
4. **備份原始圖**：留下 AI 原始輸出，方便未來調整
5. **定期更新風格**：每季評估是否需要更新圖像風格

## Quality Standard — 我的領域

### 圖片優先順序（嚴格遵守）
1. ① **真實企業照片**（最高優先，最高轉換）
2. ② **AI 模擬企業攝影**（寫實感強、有企業感）
3. ③ **IP 插畫**（品牌特色，限量輔助使用）

**絕對禁止：** 全部都是 AI 插畫 / 全部都是 Icon / 全部都是 Emoji

### 所有圖片必須具有
- 企業感（像真正的企業在用）
- 品牌感（色調符合咻咻打包）
- 故事性（畫面會說故事）
- 真實感（不過度 AI、不過度完美）

### Image Asset List（每次任務必交）
修改或新增頁面時，同步提交 Image Asset List：

| 欄位 | 說明 |
|------|------|
| 名稱 | 圖片用途名稱 |
| 尺寸 | 寬 × 高（px） |
| 用途 | 哪個頁面、哪個區塊 |
| AI Prompt | 生成用的完整 Prompt |
| 後續真人拍攝 | 是 / 否 |
| 存放路徑 | assets/images/[分類]/ |

### 資料夾結構（強制）
```
assets/images/
├── hero/         → 首頁大圖
├── services/     → 服務說明圖
├── team/         → 團隊照
├── warehouse/    → 倉庫實景
├── case-study/   → 客戶案例
└── ip/           → IP 角色
```

### 對標攝影風格
- **Nike** — 真實感、力量感、情感張力
- **DHL** — 物流專業、真實倉庫場景
- **Apple** — 產品感、極簡、留白
