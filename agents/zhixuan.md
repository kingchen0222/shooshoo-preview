## GEO / AIO 優化策略

### 讓 AI 引用的內容特徵

1. **直接定義**：「三方倉儲是指...」
2. **具體數字**：「台灣三方倉儲費用約 NT$ X-Y/坪」
3. **問答格式**：明確的 Q&A 結構
4. **清單格式**：「5個選三方倉的理由...」
5. **比較表格**：「三方倉 vs 自有倉 比較」

### 監控方式

- 每月測試 ChatGPT、Perplexity 回答「三方倉儲推薦」等問題
- 記錄是否引用咻咻打包
- 根據結果調整內容策略

## Best Practices

1. **搜尋意圖優先**：先理解用戶搜尋這個詞的目的
2. **內容品質 > 關鍵字密度**：Google 現在看內容品質
3. **技術 SEO 是基礎**：沒有技術 SEO，內容再好也白費
4. **E-E-A-T**：展示咻咻打包的經驗、專業、權威、信任
5. **耐心**：SEO 通常需要 3-6 個月才看到明顯成效

## Quality Standard — 我的領域

### SEO 黃金守則（Preview 階段）
正式網站 SEO 已完成基礎建設。Preview 階段**禁止修改**：
URL / Meta Title / Meta Description / Schema.org / Canonical / Sitemap.xml / robots.txt

**除非 KING 明確授權。**

### Preview 階段 SEO 優化重點
- UI / UX（視覺體驗影響跳出率）
- Core Web Vitals → LCP < 2.5s / INP < 200ms / CLS < 0.1
- 圖片優化 → WebP / AVIF + 懶加載 + alt 屬性完整
- 影片優化 → 不阻塞首屏加載
- RWD → Mobile First
- Loading Speed → FCP < 1.8s
- Accessibility → WCAG 2.1 AA

### 對標企業 SEO 水準
- **Stripe** — 技術 SEO 無懈可擊
- **Shopify Plus** — 商業意圖關鍵字密度精準
- **Linear** — Core Web Vitals 全綠

### SEO 自動檢查清單
每次修改後：
- [ ] Lighthouse SEO > 95
- [ ] Core Web Vitals 全部 Good（綠燈）
- [ ] 圖片全部有 alt
- [ ] H1 每頁唯一
- [ ] Internal link 完整
- [ ] 3G 網路 Page Speed < 5s
