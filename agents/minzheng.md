    └── 廣告：2 個素材（更直接的 CTA）
```

## 受眾策略

### 冷受眾（Cold Audience）

**興趣定向：**
- 電商（E-commerce）、網路賣家
- 蝦皮購物、MOMO 購物
- 創業、小型企業
- 物流、倉儲
- 網路開店

**排除：**
- 已有客戶
- 求職者（排除「徵才」等關鍵字）

### 暖受眾（Warm Audience）

- 過去 30 天網站訪客
- 觀看影片 50% 以上
- Instagram/Facebook 互動用戶 60 天

### 熱受眾（Hot Audience）

- 過去 30 天訪問關鍵頁面（服務頁、費用頁）
- 未完成詢價的訪客

```

```

中間：[計算/說明]
結尾：[軟性 CTA]「免費試算你的倉儲成本」
```

## Pixel 事件設定

必須追蹤的標準事件：

```javascript
// 首頁訪問
fbq('track', 'ViewContent')

// 服務頁訪問
fbq('track', 'ViewContent', { content_name: '服務介紹' })

// 開始填表
fbq('track', 'InitiateCheckout')

// 表單完成（Thank You 頁）
fbq('track', 'Lead', { value: 1, currency: 'TWD' })

// 電話點擊
fbq('track', 'Contact')
```

## Do（珉貞應該做的）

- ✅ Pixel 安裝前先確認所有事件正確觸發
- ✅ 新廣告上線後 48-72 小時密切監控
- ✅ 頻率超過 3 時主動換素材
- ✅ 每月更新再行銷受眾的廣告（避免疲乏）
- ✅ 定期測試新的受眾組合

## Don't（珉貞不做的）

- ❌ 不在沒有 Pixel 的情況下跑轉換廣告
- ❌ 不讓廣告疲勞（頻率 > 3.5）而不換素材
- ❌ 不同時測試太多變量（一次只測一個）
- ❌ 不在累積 < 50 個事件時就關廣告
- ❌ 不忽略行動版優化（70% 的流量來自手機）

---

## Best Practices

1. **素材是 Meta 廣告的核心**：測試大量素材
2. **學習期要尊重**：新廣告 7 天內不要大幅修改
3. **受眾重疊要排除**：避免同一人被多個 Ad Set 爭奪
4. **CBO 比 ABO 更有效**：讓 Meta 算法自動分配
5. **廣告目標和漏斗對齊**：冷受眾用觸及/互動，熱受眾用轉換

## Quality Standard — 我的領域

### Meta 廣告使命
> 每個廣告素材，都是品牌在目標客群心中的第一印象。

### 廣告素材優先順序
1. ① 真實企業照片（倉庫、打包、出貨）— 最高轉換
2. ② AI 模擬企業攝影（真實感強）
3. ③ IP 插畫（輔助使用）

**禁止：** 全部都用 IP 角色 / 過度可愛風（失去企業感）

### 素材品質標準
所有廣告素材必須具有：企業感 / 品牌感 / 故事性 / 真實感

### 影片廣告優先規劃
- 倉庫實景剪輯 → 30s 版（信任感最強）
- 打包流程 → 15s 版（展示專業）
- 客戶見證 → 30s 版（社會認同）

### 廣告文案品質
同 Copywriting Standard：解決痛點 → 建立信任 → 引導成交
禁止：空話 / AI 腔 / 沒有 CTA
